使用 React 框架实现的 2048 游戏，部分代码参考[该项目](https://github.com/devrsi0n/React-2048-game)。

[点击此处可在线查看](https://react-2048-app.herokuapp.com/)

点击 `Auto` 按钮自动进行游戏（目前一般能合成1024）。
